Interface SingleThreadSetQuery<ResultType extends SingleThreadSetQuery.Result<?,?,?>>

  • All Known Implementing Classes:
    BlockingTree, Deadlocks, TopContenders

    public interface SingleThreadSetQuery<ResultType extends SingleThreadSetQuery.Result<?,?,?>>
    Query single ThreadSet. Run query against a subset of runtime threads passed in using query(ThreadSet). Query can access whore runtime and all its threads using initialSet.getProcessRuntime(). It is up to query implementation to decide that an "initial set" mean in its context. For instance Deadlocks detect only those deadlocks where at least one of initial threads are part of the cycle.

    ogondza
